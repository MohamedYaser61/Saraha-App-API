- Explain Multer
  - Multer is a Node.js middleware for handling file uploads
  - Parses multipart/form-data requests
  - Handles single and multiple file uploads
  - Provides file filtering, size limits, and storage configuration
  - Stores files in memory or to disk via storage engines
  - Works seamlessly with Express applications
- Show how to use Multer in an Express app
  - Install Multer via npm
  - Import Multer in your Express app
  - Configure storage options (disk or memory)vironment variables
